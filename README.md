# Bingo
Bingó tábla, 
Indításhoz ajánlott a node index.js
A text.txt fájlból olvassa be hogy miket fog beletenni a tábla celláiba, ismétlődésnek nem kéne lenni, de ha van akkor a fájl utolsó sorába egy üres entert kell tenni. Ha nincs elég elem hogy ismétlődés nélkül elossza az adatokat akkor az első értéket fogja ismételni.
A tábla méretét a 'meret' nevű változóban lehet állítani, alap értéke 3, ez 3x3, 5 esetén 5x5.
A programhoz tartozik egy timer is amit az Enter-el lehet indítani és extra időt hozzáadni a '+'-al.
Az idő állítható a 'ora' változóval.
